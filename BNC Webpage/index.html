Base Number Counter

You're used to decimal (base 10), or binary if you're quirky like that (base 2). However, what if you could convert to any base number. This is where this site comes in, in which you can generate a list of not only base 10 or 2, but also 1, 3, 4, 5, and more.

(Note: If you want to look up a specific number, put that number in both min and max range)
